# LocalTools

A collection of local tools and utilities.

## Admin Panel

Access the admin panel at [https://admin.localtools.app](https://admin.localtools.app) for:

### Features
- **Subscription Management**: Manage your LocalTools subscriptions, view billing details, and update payment methods
- **Dashboard**: Get an overview of your tools usage, activity metrics, and system status
- **Basic Features**: Access core administrative functions and settings for your LocalTools account

## Getting Started

Visit the admin panel to configure and manage your LocalTools installation.
